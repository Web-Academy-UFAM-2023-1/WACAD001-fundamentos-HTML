Tags HTML

A seguir algumas tags HTML separadas por uma linha horizontal (hr).

*hr

Linha horizontal.

h1

Título principal (indicado usar somente um por página).

h2, h3, h4, h5, h6

Títulos alternativos ou subtítulos

p

Parágrafo.

a

Hiperlink.

div

Bloco genérico.

br

Quebra de linha.

*Em HTML5, a tag 'hr' define uma pausa temática. Em HTML 4.01, a tag 'hr' representa uma regra horizontal. No entanto, a tag 'hr' ainda pode ser exibida como uma regra horizontal nos navegadores visuais, mas agora é definido em termos semânticos, em vez de termos de apresentação.
